# Customer Retention & Churn Analysis
# Project Overview
This Power BI project analyzes customer retention and churn patterns to understand customer behavior and identify factors that influence customer loyalty. The dashboard provides insights that can help businesses reduce churn and improve customer retention.
# Objectives
* Analyze customer churn trends.
* Identify customer segments with higher churn rates.
* Understand customer retention patterns.
* Provide recommendations to improve customer satisfaction and loyalty.
# Tools Used
* Power BI Desktop
* Customer Churn Dataset
* DAX for calculations
# Dashboard Analysis
The dashboard includes:
* Churn Analysis by Contract Type
* Churn Analysis by Internet Service
* Gender-wise Customer Distribution
* Customer Tenure Analysis
* Monthly Charges Analysis
* Interactive Filters (Slicers)
# Key Insights
* Customers with month-to-month contracts are more likely to churn.
* Most churn occurs during the early stages of customer tenure.
* Customers with longer contracts show better retention.
* Higher monthly charges may contribute to customer churn.
# Recommendations
* Encourage customers to choose long-term contracts.
* Improve onboarding and support for new customers.
* Offer retention incentives for high-risk customers.
* Regularly monitor customer satisfaction and engagement.
# Conclusion
This dashboard helps businesses understand customer behavior, identify churn risks, and make data-driven decisions to improve customer retention and long-term growth.
